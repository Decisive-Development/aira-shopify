AIRA Shopify Store — UX/Conversion Implementation Brief
OBJECTIVE: Two user paths, zero friction. New customer → starter kit. Returning customer → refills. Every click either educates or converts.
NAV RESTRUCTURE:

Remove: "Home" (logo does this)
Keep as text links: Shop | Refills | About (with dropdown: Backed by science, Tutorials, Our story, Affiliate program)
"Shop" links to /products/aira-starter-kit (direct product page, NOT /collections/nasal-strips)
"Refills" links to /products/aira-refill-tabs (direct product page, NOT /collections)
Keep the styled orange CTA button: rename to "Shop Starter Kit" → links to /products/aira-starter-kit
Fix the current href="#" on the nav CTA — this is currently a dead link
Keep cart icon
Keep account icon

HOMEPAGE SECTION ORDER (reorder existing sections):

Hero — keep as-is structurally. Fix the "Shop now" button href from "#" to /products/aira-starter-kit. Keep "Learn more" as placeholder for now (will link to a compelling page or section later).
Before & After slider — keep. Good visual proof, earns attention early.
Why it matters (health effects tabs) — move UP from position 4. This educates on the problem before showing the product. End with a CTA: "Shop Starter Kit" → /products/aira-starter-kit
Science stats section — keep after "Why it matters". Problem → data → solution flow. End with CTA.
Why Aira (feature carousel) — this is the product solution. Keep after stats.
Reviews — social proof after features. Keep.
FAQ — keep at bottom, before footer.
Final CTA block — add a new simple section: heading + "Shop Starter Kit" button. Last chance before footer.

REMOVE FROM HOMEPAGE:

The "Our products" dual product card section (section showing both starter kit and refills). This creates a decision point that slows conversion. Homepage sells the starter kit only. Returning customers use the "Refills" nav link.

STARTER KIT PRODUCT PAGE (/products/aira-starter-kit):

Current layout is: image, title, price, variant dropdown, quantity, add to cart, buy it now, reviews. This needs expansion:
Add below the variant/price area: a "What's included" list (4 bands, 15 tabs, and whatever else is in the box)
Add a "How it works" 3-step visual section below the buy box
Add a short FAQ accordion (3-4 top questions, pulled from homepage FAQ)
Below add-to-cart: small text link "Already have a strip? Buy refill tabs →" → /products/aira-refill-tabs
Add purchase option toggle: "One-time purchase" / "Subscribe & Save X%" above add-to-cart
Implement sticky add-to-cart bar on mobile (price + "Add to Cart" always visible on scroll)
Current variant selector is a dropdown — replace with visual swatches (black circle, beige circle) since there are only 2 colour options

CART BEHAVIOUR:

Replace /cart page redirect with a slide-out cart drawer
On add-to-cart trigger: show cart drawer with upsell — "Add a refill pack" with one-click add button
If refills are already in cart, suppress the upsell

REFILLS PAGE (/products/aira-refill-tabs):

Keep minimal. No hero, no storytelling. This serves returning customers.
Quantity selector, subscription toggle ("Subscribe & save X%"), add to cart
Optionally: a small "Don't have a strip yet? Start here →" link to starter kit for anyone who lands here by mistake

COLLECTION PAGE (/collections/nasal-strips):

Remove from all navigation and CTAs
Do not delete — keep for SEO indexing
Nothing in the user flow should link to it

FOOTER (keep current structure):

Shop column: Starter Packs, Refills
Resources column: How to use, Blog, Return, FAQ
Company column: Affiliate Program, Our Story, Contact Support
Legal: Privacy Policy, Terms of Service

CMS TASKS (Shopify admin, not code):

Update nav menu links as specified above
Set up subscription selling plan via subscription app (Recharge, Loop, or Shopify native) with 30/60/90 day intervals
Configure cart upsell (via app or custom Liquid in cart drawer)
Remove /collections/nasal-strips from all navigation menus
Delete test reviews from Judge.me ("Tom2", "Tom")
